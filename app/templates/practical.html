Practical Questions

Solve the following questions in the Server Virtual Machine

  1. Set SELinux to Enforcing mode

  2. Create a group named "sysadmin" and "manager". A user sarah and natasha should belongs to "manager" group as a secondary group . A user harry should not have access to interactive shell and he should not be a member of "manager" group. passwd for all user created should be "password".

  3. Create the Directory "/home/manager" with the following characteristics. Group ownership of "/home/manager" should go to "manager" group. The directory should be have full permission for all members off "manager" group but not to any other users accept "root". Files created under "/home/manager" should get the same group ownership is set to the "manager" group

  4. The user sarah must configure a cron job that runs today at 14:23 and executes /bin/echo "hyer" and deny the max user for creating cronjob

  5. Copy file /etc/fstab to /var/tmp and configure the "ACL" as mentioned :- The file /var/tmp/fstab is owned by the "root". The file /var/tmp/fstab belongs to the group "root". The file /var/tmp/fstab should not be executable by other's. User "sarah" should able to read and write to the file. User "natasha" can neither read nor write to the file. Other users (future and current) should be able to read /var/tmp/fstab.

  6. Create the user "dax" with uid 4223
